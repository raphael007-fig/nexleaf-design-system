# OptionCard

A selectable illustrated card for one-of-N decisions where each option deserves real
explanation — media + title + description, with radio semantics. Born as the
monitoring-method chooser in the ColdTrace Add Equipment flow (Figma 8055-205358).

<Canvas of={OptionCardStories.Interactive} />

---

## When to use

- **OptionCard** — a small set (2–4) of rich, mutually exclusive choices that steer a flow
- **RadioGroup** — compact one-line choices inside a form
- **OptionList** — searchless pick lists inside popovers/panels

## States

Rest · hover (primary ring) · selected (`bg-selected` + 2px primary ring) · disabled.
Cards are keyboard-operable (`role="radio"`, Enter/Space). Wrap the set in
`role="radiogroup"` with a label.
