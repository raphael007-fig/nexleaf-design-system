# Tabs

Horizontal navigation for switching between related views within the same page context. Does not cause full page navigation — it filters or switches the content below.

<Canvas of={TabsStories.AllStates} />

---

## When to use

- Filtering a list by status (All / Active / Archived)
- Switching between related data views (Overview / Details / History)
- Mobile: larger 14px tabs for touch targets

Do **not** use Tabs for top-level page navigation — use the sidebar/nav instead.

```html
  <button role="tab" aria-selected="true" class="nx-tab nx-tab--active">

    <span class="nx-tab__badge">24</span>
  </button>
  <button role="tab" aria-selected="false" class="nx-tab">
    Active
    <span class="nx-tab__badge">12</span>
  </button>

  <button role="tab" class="nx-tab nx-tab--active">Morning</button>
```

## Accessibility

- Wrap in `<div role="tablist">`
- Each tab: `role="tab"`, `aria-selected="true/false"`
- The content panel: `role="tabpanel"` with `aria-labelledby` pointing to the active tab's `id`
- Disabled tabs: add `aria-disabled="true"` (the React component handles this automatically when `disabled: true`)

### Keyboard support

Follows the WAI-ARIA Authoring Practices [tablist pattern](https://www.w3.org/WAI/ARIA/apg/patterns/tabs/). Implements a roving-tabindex: only the active tab is in the page's tab sequence, and arrow keys move focus + selection between siblings.

| Key | Action |
|-----|--------|
| `Tab` | Move focus into / out of the tablist. Only the active tab is reachable from outside. |
| `Arrow Right` | Move focus and selection to the next non-disabled tab (wraps to the first). |
| `Arrow Left` | Move focus and selection to the previous non-disabled tab (wraps to the last). |
| `Home` | Move focus and selection to the first non-disabled tab. |
| `End` | Move focus and selection to the last non-disabled tab. |
| `Enter` / `Space` | Activate the focused tab. (Arrow keys already activate, so these are only needed if you wire up a manual-activation variant later.) |

Disabled tabs are skipped by arrow / Home / End navigation but remain in the DOM for screen reader context.

## Angular component

```ts
import { NxTabsModule } from 'components/Tabs/tabs.module';

<nx-tabs
  [activeIndex]="active"
  (change)="active = $event"
  [tabs]="[
    { label: 'All', badge: 24 },
    { label: 'Active', badge: 12 },
    { label: 'Archived', disabled: true }

/>

// Fitted
<nx-tabs [fitted]="true" [activeIndex]="active" (change)="active = $event"
  [tabs]="[{ label: 'Morning' }, { label: 'Evening' }]" />
```

## React component

### Selection callback API

The canonical callback is **`onSelect(id, item, index)`** — it matches the rest of the component library and gives consumers the full item plus its index without a lookup. `id` is the tab's `id` (falling back to the string-coerced index when omitted). The 3rd `index` arg is there for callers that track selection by position.

```jsx
// id-based — matches every navigation/structural component
<Tabs
  activeIndex={active}
  onSelect={(id, item) => navigate(`/orders?status=${id}`)}
  tabs={[
    { id: 'all',    label: 'All' },
    { id: 'active', label: 'Active' },
    { id: 'draft',  label: 'Draft' },
  ]}
/>

// index-based — read the 3rd arg when you track position
<Tabs activeIndex={active} onSelect={(_id, _item, i) => setActive(i)} tabs={…} />
```

If `item.id` is omitted, `onSelect` falls back to passing the string-coerced index (`'0'`, `'1'`, …) as the first argument.

```jsx
import { Tabs } from 'components/Tabs/Tabs.jsx';

const [active, setActive] = useState(0);

<Tabs
  activeIndex={active}
  onSelect={(_id, _item, i) => setActive(i)}
  tabs={[
    { label: 'All', badge: 24 },
    { label: 'Active', badge: 12 },
    { label: 'Archived', disabled: true },
  ]}
/>

// Fitted (equal width)
<Tabs fitted activeIndex={active} onSelect={(_id, _item, i) => setActive(i)}
  tabs={[{ label: 'Morning' }, { label: 'Evening' }]} />

// With "More views" and "Add new" controls
<Tabs moreViews canAddNew activeIndex={active} onSelect={(_id, _item, i) => setActive(i)}
  tabs={[{ label: 'All' }, { label: 'Active' }]} />
```
