# MetricCard

Displays a high-level operational KPI in a compact, scannable card. Can function as a static informational display or an interactive dashboard filter that narrows visible content when selected.

<Canvas of={MetricCardStories.AllStates} />

---

## When to use

- Monitoring dashboards with key operational metrics
- Equipment management — offline count, alert totals, installation status
- Dashboard filter bars where each card narrows data when clicked
- Analytics overviews that need to be scannable at a glance

**Avoid** when content is text-heavy, requires detailed interpretation, or when a table would provide better clarity.

## States

| State | Trigger | Background |
|-------|---------|------------|
| Rest | Default | `#ffffff` |
| Hover | `onMouseEnter` (clickable only) | `#f7f7f7` |
| Active / Pressed | `onMouseDown` | `#f3f3f3` + inset shadow |
| Selected | `selected={true}` | `rgba(0,0,0,0.08)` |
| Focus | Keyboard focus (clickable only) | `#ffffff` + `2px #005bd3` ring |
| Disabled | `disabled={true}` | `rgba(0,0,0,0.05)` + muted text |
| Loading | `loading={true}` | Skeleton bars, `#e8e8e8` |

## Anatomy

1. **Container** — `border-radius: 12px`, `padding: 16px`, `gap: 12px`
2. **Metric label** — 13px / 650 weight / `#616161`
3. **Info icon** — 14px ⓘ icon, optional click handler
4. **Metric value** — 30px / 700 weight / `#303030` / `letter-spacing: -0.3px`
5. **Badge** — optional, 20px height, tone-colored pill

## Accessibility

- When `onClick` is provided, the card renders as `role="button"` with `tabIndex={0}`
- Keyboard: Enter and Space activate the card
- Selected state is communicated via `aria-pressed` (recommended in consuming pages)
- Info icon has `aria-label="More information"`
- Disabled cards are non-interactive — do not use `disabled` HTML attribute on the wrapper (it would break keyboard navigation for screen readers)

## Angular component

```ts
import { NxMetricCardModule } from 'components/MetricCard/metric-card.module';

// Static display
<nx-metric-card
  title="Total Installations"
  metric="247"
  [badge]="{ label: '3 Critical', tone: 'critical' }"
/>

// Interactive filter card
<nx-metric-card
  title="Devices Offline"
  metric="12"
  [badge]="{ label: '12 Offline', tone: 'warning' }"
  [selected]="activeFilter === 'offline'"
  (click)="toggleFilter('offline')"
/>

// Loading / Disabled
<nx-metric-card [loading]="true" />
<nx-metric-card title="Unknown" metric="—" [disabled]="true" />
```

## React component

```jsx
import { MetricCard } from 'components/MetricCard/MetricCard.jsx';

// Static display
<MetricCard
  title="Total Installations"
  metric="247"
  badge={{ label: '3 Critical', tone: 'critical' }}
/>

// Interactive filter card
<MetricCard
  title="Devices Offline"
  metric="12"
  badge={{ label: '12 Offline', tone: 'warning' }}
  selected={activeFilter === 'offline'}
  onClick={() => setActiveFilter(prev => prev === 'offline' ? null : 'offline')}
/>

// With info icon tooltip
<MetricCard
  title="Active Alerts"
  metric="5"
  badge={{ label: '2 High Priority', tone: 'critical' }}
  onInfoClick={() => setTooltipOpen(true)}
/>

// Loading
<MetricCard loading />

// Disabled
<MetricCard
  title="Unknown Devices"
  metric="—"
  badge={{ label: 'No data', tone: 'neutral' }}
  disabled
/>
```
