# RadioButton

A circular selection control representing a single choice from a mutually exclusive list. Pair multiple RadioButtons inside a `RadioGroup` so only one can be active.

<Canvas of={RadioStories.AllStates} />

---

## When to use

- **RadioButton** — mutually exclusive choice between two or more options where all options should be visible
- **Checkbox** — opt-in agreement or independent boolean choices within a form
- **SelectInput** — mutually exclusive choice when there are many options and space is constrained
- **Toggle** — single on/off setting that takes effect immediately

Always present radios in groups of two or more — never a single radio.

## States

| State | Ring (off) | Ring (on) | Notes |
|-------|------------|-----------|-------|
| Rest | `#fdfdfd` + `0.66px solid #8a8a8a` | `#303030` | Inner dot `#ffffff`, `6×6px` |
| Hover | `#fafafa` + `0.66px solid #616161` | — | Slightly darker border |
| Focus | rest colours | rest colours | + `0 0 0 2px #005bd3` ring |
| Disabled | `rgba(0,0,0,0.08)` | `rgba(0,0,0,0.08)` | Inner dot `rgba(160,160,160,0.7)`, `cursor: not-allowed` |
| Error (off) | `#fee9e8` + `0.66px solid #8e1f0b` | — | Shows error message below |
| Error (on) | — | `#8e1f0b` | Shows error message below |
| Magic — Rest | `#f8f7ff` + `0.66px solid #9474ff` | `#8051ff` | Label `#5700d1` when checked |
| Magic — Hover | `#f3f1ff` + `0.66px solid #8051ff` | — | — |

**Outer ring** — `16×16px` circle. **Inner dot** — `6×6px`, only rendered when `checked`.

## Accessibility

- Uses `role="radio"` and `aria-checked` to communicate state to screen readers
- Keyboard accessible — `Space` and `Enter` select the radio
- Disabled radios have `tabIndex={-1}` and `cursor: not-allowed`
- Use the `RadioGroup` wrapper to share a `name` and provide a group `title` — screen readers will announce the group label
- Always present at least two options — a single radio should be a Checkbox instead

## Angular component

```ts
import { NxRadioModule } from 'components/RadioButton/radio.module';

// Basic
<nx-radio-group [(value)]="pref">
  <nx-radio value="email" label="Email" />
  <nx-radio value="sms"   label="SMS" />
  <nx-radio value="push"  label="Push notification" />
</nx-radio-group>

// With help text
<nx-radio value="email" label="Email" helpText="Daily digest at 8am" />

// Magic tone
<nx-radio value="pro" label="Pro" tone="magic" />

// Error
<nx-radio-group [(value)]="plan" error="Please select a plan.">
  <nx-radio value="basic" label="Basic" />
  <nx-radio value="pro"   label="Pro" />
</nx-radio-group>

// Disabled
<nx-radio value="none" label="None" [disabled]="true" />
```

## React component

```jsx
import { RadioButton, RadioGroup } from 'components/RadioButton/RadioButton.jsx';

// Basic group
<RadioGroup
  title="Notification preference"
  name="pref"
  value={pref}
  onChange={setPref}
  options={[
    { value: 'email', label: 'Email', helpText: 'Daily digest at 8am' },
    { value: 'sms',   label: 'SMS' },
    { value: 'push',  label: 'Push notification' },
  ]}
/>

// Magic tone
<RadioGroup
  title="Choose a plan"
  required
  name="plan"
  value={plan}
  onChange={setPlan}
  tone="magic"
  options={[
    { value: 'basic', label: 'Basic' },
    { value: 'pro',   label: 'Pro' },
  ]}
  error="Please select a plan."
/>

// Single radio (composed manually)
<RadioButton label="Email" checked={v === 'email'} onChange={() => setV('email')} />

// Disabled
<RadioButton label="None" checked={false} disabled />
```
