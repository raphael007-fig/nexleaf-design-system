# DatePicker

A visual calendar for picking a single date or a range of dates. Supports single-month, multi-month side-by-side, and vertically-stacked (mobile) layouts.

<Canvas of={DatePickerStories.AllStates} />

---

## When to use

- **DatePicker** ‚Äî picking a specific calendar date or range (booking, scheduling, filtering)
- **NumberInput / TextInput** ‚Äî when only a year, month, or day-of-month is needed
- **SelectInput** ‚Äî when the user is choosing between a small number of preset dates

Use `multiMonth` for date-range selection on desktop, and `verticalStack` to reflow for mobile.

## Anatomy

| Part | Style |
|------|-------|
| Card | `#fdfdfd` bg, `0.66px solid #e0e0e0` border, `12px` radius, `16px` padding |
| Month header | `13px` `#303030`, weight `500`, centered |
| Prev/Next arrows | 16px arrows in `#303030`, 32√ó32px tap target |
| Weekday row | `12px` `#9e9e9e`, weight `500`, 32√ó32px cells |
| Day ‚Äî current month | `13px` `#303030` |
| Day ‚Äî other month | `13px` `#b5b5b5` |
| Day ‚Äî hover | `#f5f5f5` bg, full radius |
| Day ‚Äî selected | `#303030` bg, `#ffffff` text, full radius (or capped if range edge) |
| Day ‚Äî range middle | `#ebebeb` bg, `#303030` text, no radius |
| Day ‚Äî today (unselected) | inset `0.66px solid #8a8a8a` ring |
| Day ‚Äî disabled | `#b5b5b5` text, no hover |

Day cells are `32√ó32px`. Range edges use half-radius (`999px 0 0 999px` on the start, `0 999px 999px 0` on the end) so the highlighted middle row reads as one continuous bar.

## Accessibility

- Day cells are reachable via keyboard ‚Äî `Tab` enters the grid, arrow keys move between days
- Selected day has `aria-selected="true"`
- Disabled days have `aria-disabled="true"` and skip tab order
- Month title is announced via `aria-live="polite"` when navigation changes the cursor
- Pair with a visible label outside the picker (e.g., "Departure date") when used in a form
- Range mode: announce the start once chosen, then prompt for the end

## React component

```jsx
import { DatePicker } from 'components/DatePicker/DatePicker.jsx';

// Single date
<DatePicker value={date} onChange={setDate} />

// Range
<DatePicker value={range} onChange={setRange} allowRange />
// value shape: { start: Date | null, end: Date | null }

// Multi-month for range selection
<DatePicker value={range} onChange={setRange} allowRange multiMonth />

// Mobile (vertically stacked months)
<DatePicker value={range} onChange={setRange} allowRange multiMonth verticalStack />

// Bounded
<DatePicker
  value={date}
  onChange={setDate}
  minDate={new Date(2024, 0, 1)}
  maxDate={new Date(2024, 11, 31)}
/>

// Custom initial month (when value is null)
<DatePicker value={null} onChange={setDate} initialMonth={new Date(2024, 5, 1)} />
```

**Props**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `Date \| { start, end } \| null` | ‚Äî | Selected date (single) or range object |
| `onChange` | `(value) => void` | ‚Äî | Fires with new date or `{ start, end }` |
| `allowRange` | `boolean` | `false` | Enables range selection |
| `multiMonth` | `boolean` | `false` | Show two months side by side |
| `verticalStack` | `boolean` | `false` | Stack multi-month vertically (mobile) |
| `minDate` | `Date` | ‚Äî | Earliest selectable date |
| `maxDate` | `Date` | ‚Äî | Latest selectable date |
| `initialMonth` | `Date` | today | First month shown when there's no value |
