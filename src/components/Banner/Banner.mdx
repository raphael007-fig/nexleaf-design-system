# Banner

Full-width contextual message. Communicates feedback, warnings, or important information about a page or action. Not used for toast/snackbar notifications.

<Canvas of={BannerStories.AllStates} />

---

## When to use

- **Info** — neutral guidance, reminders, or non-critical notices
- **Success** — confirmation that an action completed (form saved, record created)
- **Warning** — something needs attention but isn't blocking
- **Critical** — an error occurred or a required action is blocked

Use the **titled form** (with a header bar) for longer messages that need a title and body copy, or when actions are included. Use the **simple form** for short one-liners.

---

## Three visual forms

### Simple
White card with a colored icon pill (28×28, tone color, 8px radius), message text, optional actions row, and optional × dismiss button.

### Titled
Two-part card: full-width colored header bar (tone color + icon + title, optional dismiss) + white body with message and action buttons.

### In-card (compact)
Tinted background using the tone's light color, inline icon, message, and optional actions. Used inside cards and panels where space is constrained.

| Tone | Header / icon pill bg | Header text | In-card bg | In-card text | Usage |

## CSS selector structure

```css
@import 'tokens/tokens.css';
@import 'components/Banner/Banner.css';
```

```html
<!-- Simple info -->
<div class="nx-banner nx-banner--info" role="status">
  <!-- info icon svg -->
  <span class="nx-banner__text">Your session expires in 10 minutes.</span>
</div>

<!-- Simple with dismiss -->
<div class="nx-banner nx-banner--warning" role="alert">
  <!-- warning icon -->
  <span class="nx-banner__text">This reading is outside the expected range.</span>
  <button class="nx-banner__dismiss" aria-label="Dismiss"><!-- × icon --></button>
</div>

<!-- Titled critical -->
<div class="nx-banner--titled" role="alert">
  <div class="nx-banner__header nx-banner__header--critical">
    <!-- critical icon -->
    <span>Unable to save reading</span>
    <button class="nx-banner__dismiss" aria-label="Dismiss"><!-- × icon --></button>
  </div>
  <div class="nx-banner__body">
    <p>The temperature value is required. Please enter a valid reading before submitting.</p>
    <div class="nx-banner__actions">
      <button class="nx-btn nx-btn--secondary">Review fields</button>
    </div>
  </div>
</div>
```

| `--nx-font-weight-bold` | `600` | Titled banner header title |
| `--nx-text-default` | `#303030` | Titled banner body text |
| `--nx-space-3` | `12px` | Margin above action buttons |

## Accessibility

- Simple banners that appear in response to user action: use `role="status"` (polite) or `role="alert"` (assertive/error)
- Titled critical banners: always `role="alert"` so screen readers announce immediately
- Dismiss button must have an accessible name: `aria-label="Dismiss"` or visually hidden text
- Do not auto-dismiss error banners — the user must acknowledge them

## Angular component

```ts
import { NxBannerModule } from 'components/Banner/banner.module';

// Simple
<nx-banner tone="info">Your session expires in 10 minutes.</nx-banner>
<nx-banner tone="success">Reading saved successfully.</nx-banner>
<nx-banner tone="warning" [dismissable]="true">Value is outside expected range.</nx-banner>

// Titled
<nx-banner
  tone="critical"
  title="Unable to save"
  [dismissable]="true"
  [actions]="[{ label: 'Review', onClick: fn }]"
>
  Please correct the highlighted fields and try again.
</nx-banner>

// In-card
<nx-banner tone="info" [inCard]="true">Your session expires in 10 minutes.</nx-banner>
```

## React component

```jsx
import { Banner } from 'components/Banner/Banner.jsx';

// Simple
<Banner tone="info">Your session expires in 10 minutes.</Banner>
<Banner tone="success">Reading saved successfully.</Banner>
<Banner tone="warning" dismissable>Value is outside expected range.</Banner>

// Titled (colored header + white body)
<Banner
  tone="critical"
  title="Unable to save"
  dismissable
  actions={[{ label: 'Review', onClick: fn }]}
>
  Please correct the highlighted fields and try again.
</Banner>

// In-card (compact tinted variant — use inside cards/panels)
<Banner tone="info" inCard>Your session expires in 10 minutes.</Banner>
<Banner tone="warning" inCard dismissable>Value is outside the expected range.</Banner>
<Banner tone="critical" inCard actions={[{ label: 'Fix now', onClick: fn }]}>
  Temperature reading is missing.
</Banner>
```
