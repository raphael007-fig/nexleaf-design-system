# Accordion

A collapsible section that hides or reveals content on demand. Used to reduce visual complexity in long forms and detail views.

<Canvas of={AccordionStories.AllStates} />

---

## When to use

- Form sections that are optional or secondary (Morning / Evening readings)
- Details panels where most users only need to see one section at a time
- Settings or configuration grouped by topic

Do **not** use Accordion when all sections need to be visible simultaneously — use a flat form layout instead.

## CSS selector structure

```css
@import 'tokens/tokens.css';
@import 'components/Accordion/Accordion.css';
```

```html
<!-- Closed -->
<div class="nx-accordion">
  <button class="nx-accordion__trigger" aria-expanded="false">
    <span class="nx-accordion__title">Morning Reading</span>
    <svg class="nx-accordion__chevron"><!-- chevron path --></svg>
  </button>
</div>

<!-- Open -->
<div class="nx-accordion">
  <button class="nx-accordion__trigger nx-accordion__trigger--open" aria-expanded="true">
    <span class="nx-accordion__title">Morning Reading</span>
    <svg class="nx-accordion__chevron"><!-- chevron path --></svg>
  </button>
  <div class="nx-accordion__body">
    <!-- form fields -->
  </div>
</div>

<!-- Required + has content (closed) -->
<div class="nx-accordion">
  <button class="nx-accordion__trigger" aria-expanded="false">
    <span class="nx-accordion__title nx-accordion__title--required">
      Morning Reading
      <span class="nx-accordion__dot"></span>
    </span>
    <svg class="nx-accordion__chevron"><!-- chevron path --></svg>
  </button>
</div>
```

## Accessibility

- Trigger is a `<button type="button">` — keyboard navigable, announced by screen readers
- Use `aria-expanded="true/false"` on the trigger to communicate open state
- The body content must be hidden from assistive tech when closed — either use `hidden` attribute, `display: none`, or control `visibility`
- `aria-controls` on the trigger pointing to the body `id` is recommended for robust AT support

<Accordion
  title="Morning Reading"
  required
  open={open}
  onToggle={() => setOpen(o => !o)}
  hasContent={!!morningValue}
