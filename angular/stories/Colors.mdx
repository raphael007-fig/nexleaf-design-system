# Color Tokens

Every color in the system maps to a `--nx-*` custom property from `tokens.css`
(shipped inside `@nexleaf/angular/styles.css` and standalone as
`@nexleaf/angular/tokens.css`). **Never hardcode hex values** — reference the
token. Swatches below render from the live CSS variables, so they are always
exactly what your app gets.

export const Swatch = ({ token, note }) => (
  <div style={{ display: 'flex', alignItems: 'center', gap: 12, padding: '6px 0' }}>
    <div style={{ width: 44, height: 32, borderRadius: 6, border: '1px solid #e0e0e0', background: `var(${token})`, flexShrink: 0 }} />

    <span style={{ fontSize: 13, color: '#616161' }}>{note}</span>
  </div>
);
